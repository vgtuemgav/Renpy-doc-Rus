Шаблонные проекты
=================

Можно создать шаблонный проект, который можно будет использовать для создания
новых проектов. Это можно использовать, чтобы облегчить создателям начало
разработки игры, например, с разработанным вами GUI, а не с GUI по
умолчанию, поставляемым с Ren'Py.

Если на компьютере создателя присутствует хотя бы один шаблонный проект,
при создании новой игры ему будет предложено выбрать шаблонный проект.
Если он это сделает, шаблонный проект будет скопирован в новый проект,
и будут внесены следующие изменения:

* Переменные :var:`config.name`, :var:`config.save_directory` и
  :var:`build.name` будут установлены в имя нового
  проекта.

* Каталог tl будет удалён и заменён переводами по умолчанию,
  взятыми из лаунчера.

* Если будут найдены переводимые строки или комментарии, совпадающие
  с таковыми в лаунчере, переводы будут скопированы в новый проект.

Обычно это является приемлемой отправной точкой для новых проектов.


Создание шаблонного проекта
---------------------------

Чтобы создать шаблонный проект, вам нужно создать проект Ren'Py, а затем
добавить в него файл project.json. Этот файл должен содержать следующий
текст::

    { "type" : "template" }

Сохраните его, затем обновите лаунчер, и вы сможете использовать
свой шаблон.


Распространение шаблонного проекта
----------------------------------

Вы можете распространять шаблонный проект, заархивировав его вручную. (Вы
не должны использовать команду Distribute.) Может иметь смысл удалить
каталоги ``game/saves`` и ``game/cache`` перед архивацией,
хотя это не обязательно.

Конечный пользователь может разархивировать шаблонный проект в свой
каталог проектов, а затем обновить лаунчер, чтобы сделать его
доступным.
