=====================
Внутриигровые покупки
=====================

Ren'Py включает в себя высокоуровневый фреймворк для внутриигровых покупок.
Этот фреймворк в настоящее время поддерживает только покупки типа «разблокировка» из Apple App Store,
Google Play и Amazon Appstore.

Использование этого фреймворка довольно простое и сводится к следующим
функциям.

* На этапе `init` вашей игры зарегистрируйте доступные покупки с помощью
  функции :func:`iap.register`.
* После запуска игры проверьте, была ли совершена покупка,
  с помощью функции :func:`iap.has_purchased`.
* Позвольте пользователю приобрести товар с помощью функции :func:`iap.purchase`
  или действия :func:`iap.Purchase`.
* Позвольте пользователю восстановить покупки, приобретённые на других устройствах, с помощью
  функции :func:`iap.restore` или действия :func:`iap.Restore`.

Чтобы предлагать внутриигровые покупки, эти покупки (включая связанную с ними цену)
должны быть настроены на различных торговых площадках.
