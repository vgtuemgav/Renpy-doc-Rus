=======
Лаунчер
=======

Лаунчер Ren'Py в основном задокументирован в других разделах руководства. Эта
страница содержит информацию о лаунчере, которая не вписывается в другие места.


Папки проектов
--------------

Лаунчер поддерживает папки. Вы можете создать папку, создав директорию
внутри директории проектов, переместив проект в неё, а затем нажав «Refresh» (Обновить).
Поддерживается только один уровень вложенности папок, поэтому вы не можете создать папку внутри папки.

Папки можно разворачивать и сворачивать, нажимая на их название в лаунчере.

.. warning::

    В настоящее время лаунчер Ren'Py поддерживает только один проект с заданным именем. Поэтому невозможно
    иметь проекты с одинаковыми именами в разных папках.

Projects.txt
------------

Если в директории проектов существует файл с именем ``projects.txt``, ожидается,
что он будет содержать список полных путей, по одному на строку. Эти пути
обрабатываются так, как если бы они находились внутри директории проектов, и проекты, которые они содержат, отображаются
в лаунчере.


Без ссылок в лаунчере
---------------------

Если в базовой директории Ren'Py (той, где находятся
renpy.exe, renpy.sh и renpy.app) существует файл с именем ``no_launcher_links.txt``, лаунчер
отключит ссылки на renpy.org. Хотя это
и не гарантируется, данная функция предназначена для использования в образовательной и домашней среде, чтобы юные
пользователи Ren'Py не видели спонсоров с контентом для взрослых.
