Переменные окружения
====================

Ren'Py принимает ряд переменных окружения, которые влияют на его поведение.
Эти переменные окружения могут исчезать или изменяться в разных версиях Ren'Py.

Следующие переменные окружения управляют поведением Ren'Py:

.. envvar:: RENPY_DEBUG_SOUND

    Если установлена, Ren'Py будет генерировать исключения при возникновении ошибок аудио.

.. envvar:: RENPY_DISABLE_JOYSTICK

    Если установлена, определение джойстика будет отключено. Используйте эту переменную, если неисправный джойстик вызывает нежелательное продвижение по игре.

.. envvar:: RENPY_DISABLE_FULLSCREEN

    Если установлена, Ren'Py откажется переходить в полноэкранный режим.

.. envvar:: RENPY_DISABLE_SOUND

    Эта переменная предотвращает воспроизведение звука. Если она содержит
    "pss", воспроизведение звука будет отключено.

.. envvar:: RENPY_DRAWABLE_RESOLUTION_TEXT

    Если установлена в 0, Ren'Py не будет отрисовывать текст в разрешении экрана.

.. envvar:: RENPY_EDIT_PY

    Путь к файлу .edit.py, который указывает Ren'Py, как вызывать текстовый редактор.
    Смотрите :doc:`editor` для получения дополнительной информации.

.. envvar:: RENPY_GL_ENVIRON

    Устанавливает окружение текстур OpenGL.

.. envvar:: RENPY_GL_RTT

    Устанавливает метод рендеринга в текстуру (render-to-texture) OpenGL.

.. envvar:: RENPY_GL_VSYNC

    Эта переменная определяет, будет ли Ren'Py пытаться синхронизироваться с вертикальной разверткой дисплея.
    (Это предотвращает разрывы изображения, ценой возможного снижения частоты кадров).
    Установите её в "0", чтобы отключить синхронизацию, или в "1" для синхронизации с каждой вертикальной разверткой.

.. envvar:: RENPY_LANGUAGE

    Если установлена, указывает язык перевода, который будет использовать Ren'Py.

.. envvar:: RENPY_LESS_MEMORY

    Эта переменная заставляет Ren'Py сократить использование памяти в обмен на снижение скорости.

.. envvar:: RENPY_LESS_MOUSE

    Эта переменная заставляет Ren'Py полностью отключить мышь.

.. envvar:: RENPY_LESS_PAUSES

    Эта переменная заставляет Ren'Py отключить паузы, создаваемые текстовыми тегами {p} и {w}.

.. envvar:: RENPY_LESS_UPDATES

    Эта переменная заставляет Ren'Py уменьшить количество обновлений экрана.

.. envvar:: RENPY_LOG_EVENTS

    Если установлена, Ren'Py будет записывать события в стиле pygame в файл log.txt. Это негативно скажется на производительности, но может быть полезно для отладки определенных проблем.

.. envvar:: RENPY_MULTIPERSISTENT

    Путь к каталогу, где Ren'Py хранит данные multipersistent.

.. envvar:: RENPY_NO_STEAM

    Если присутствует в окружении, Ren'Py не будет инициализировать Steamworks.

.. envvar:: RENPY_OPEN_FILE_ENCODING

    Устанавливает кодировку по умолчанию для :func:`renpy.open_file`. В основном используется
    для помощи играм на Python 2 в запуске на версиях Ren'Py, основанных на Python 3.
    Попробуйте значения "utf-8" и "latin-1", чтобы проверить, поможет ли это запустить такие игры. (Но не забудьте вернуть значение обратно для новых игр.)

.. envvar:: RENPY_PATH_TO_SAVES

    Путь к каталогу, где Ren'Py хранит свои сохранения. Сохранения для конкретной
    игры хранятся в её подкаталоге внутри этой директории.

.. envvar:: RENPY_SCREENSHOT_PATTERN

    Шаблон для создания имён файлов снимков экрана. Он должен содержать одну
    подстановку %d. Например, установка этой переменной в "screenshot%04d.jpg"
    заставит Ren'Py записывать снимки экрана в формате JPEG, а не в обычном PNG.

    По умолчанию это относительный путь от базового каталога игры. Можно использовать абсолютный путь,
    чтобы разместить снимки экрана в другом месте файловой системы.

.. envvar:: RENPY_SEARCHPATH

    Если установлена, это список дополнительных путей, разделенных двойным двоеточием (\:\:), которые
    добавляются в :var:`config.searchpath`.

.. envvar:: RENPY_SIMPLE_EXCEPTIONS

    Если установлена, отключает графическую обработку исключений Ren'Py.

.. envvar:: RENPY_SKIP_MAIN_MENU

    Если установлена, пропускает главное меню.

.. envvar:: RENPY_SKIP_SPLASHSCREEN

    Если установлена, пропускает заставку.

.. envvar:: RENPY_SOUND_BUFSIZE

    Эта переменная управляет размером звукового буфера. Значения больше стандартного (2048)
    могут предотвратить прерывание звука ценой большей задержки между вызовом
    звука и его воспроизведением.

.. envvar:: RENPY_TEST_VC_REVISION

    Используется как значение по умолчанию для :var:`_test.vc_revision`, являющейся частью
    фреймворка автоматизированного тестирования.

.. envvar:: RENPY_TIMEWARP

    Эту переменную можно установить, чтобы заставить время идти быстрее или медленнее. Например, установка
    timewarp в 0.5 заставит всё работать на половинной скорости, а timewarp
    2.0 заставит всё работать с удвоенной скоростью.

.. envvar:: RENPY_USE_DRAWABLE_RESOLUTION

    Если установлена в 0, Ren'Py будет выполнять определённые операции (включая
    переходы-растворения и рендеринг текста) в виртуальном разрешении игры,
    а не в нативном разрешении экрана.

.. envvar:: RENPY_VARIANT

    Эта переменная должна быть установлена в список вариантов экрана, разделённых пробелами,
    которые Ren'Py должен будет использовать.

Поскольку Ren'Py использует SDL, его поведением также можно управлять с помощью
переменных окружения SDL.

При запуске Ren'Py ищет в своей директории (той, что содержит
renpy.exe или renpy.py) файл :file:`environment.txt`. Если файл существует, он будет
выполнен как Python-файл, и значения, определённые в нём, будут
использованы как значения по умолчанию для переменных окружения.
