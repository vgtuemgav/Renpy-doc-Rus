Переменные настроек

Переменные настроек хранят значения настроек Ren'Py. В то время как значение настройки должно устанавливаться во время выполнения с помощью действия Preference(), переменные настроек следует использовать вместе с оператором default для установки значения настройки по умолчанию.

Например:

default preferences.text_cps = 40

устанавливает скорость текста по умолчанию на 40 символов в секунду. Оператор default устанавливает значение настройки, только если значение по умолчанию изменилось с момента последней установки настройки. Например, если игрок изменяет скорость на 50, она останется на 50 при последующих запусках игры. Если в обновлении значение по умолчанию будет установлено на 42, настройка игрока изменится на 42. (Затем игрок может изменить ее снова.)

preferences.afm_after_click = False

Если True, режим автопрокрутки продолжится после клика. Если False, клик завершит режим автопрокрутки. Эквивалент настройки «автопрокрутка после клика».

preferences.afm_enable = False

Если True, автопрокрутка включена, иначе False. Эквивалент настройки «время автопрокрутки».

preferences.afm_time = 15

Время ожидания для режима автопрокрутки. Большие числа означают меньшую скорость, хотя преобразование в реальное время сложно, так как скорость учитывает длину строки. Эквивалент настройки «автопрокрутка».

preferences.desktop_rollback_side = "disable"

На настольной платформе прикосновения или клики по этой стороне окна вызывают откат (rollback). Одно из: "left", "right" или "disable". Это эквивалент настройки «сторона отката» на настольной платформе.

preferences.mobile_rollback_side = "disable"

На мобильной платформе прикосновения или клики по этой стороне окна вызывают откат (rollback). Одно из: "left", "right" или "disable". Это эквивалент настройки «сторона отката» на мобильной платформе.

preferences.language = None

Язык, который игрок выбрал для использования при запуске игры. Это None для языка по умолчанию или строка, содержащая язык, на который переведена игра.

Это можно использовать для установки языка по умолчанию и считывать для определения текущего языка. Действие Language() можно использовать для изменения языка.

См. Перевод для получения дополнительной информации.

preferences.emphasize_audio = False

Если True, Ren'Py будет выделять аудиоканалы, указанные в config.emphasize_audio_channels, уменьшая громкость других каналов. (Например, уменьшая громкость музыки, когда звучит голос.) Если False, этого не происходит.

preferences.fullscreen = False

Это True, когда Ren'Py работает в полноэкранном режиме, и False, когда он работает в окне. Эквивалент настройки «отображение».

preferences.gl_framerate = None

Это либо целое число, либо None. Если не None, это целевая частота кадров, которой Ren'Py будет пытаться достичь. Если это значение установлено низко (например, 30), на мониторе с высокой частотой обновления (скажем, 60 кадров в секунду), Ren'Py будет рисовать только каждый второй кадр.

Если None, Ren'Py будет пытаться рисовать с полной частотой кадров монитора.

preferences.gl_powersave = True

Определяет, как часто Ren'Py будет перерисовывать неизменный экран. Если True, Ren'Py будет рисовать экран только 5 раз в секунду. Если False, он всегда будет рисовать с максимально возможной частотой кадров.

preferences.gl_tearing = False

Определяет, является ли разрыв изображения (True) или пропуск кадров (False) предпочтительным поведением, когда игра не успевает за предполагаемой частотой кадров.

preferences.mouse_move = True

Если True, мышь будет автоматически перемещаться к выбранной кнопке. Если False, этого не произойдет. Эквивалент настройки «автоматическое перемещение».

preferences.show_empty_window = True

Если True, операторы window show и window auto будут функционировать. Если False, эти операторы отключены. Эквивалент настройки «показывать пустое окно».

preferences.skip_after_choices = False

Если True, пропуск возобновится после выбора. Если False, выбор помешает Ren'Py пропускать. Эквивалент настройки «после выбора».

preferences.skip_unseen = False

Когда True, Ren'Py будет пропускать весь текст. Когда False, Ren'Py будет пропускать только текст, который был прочитан игроком в любой сессии. Эквивалент настройки «пропуск».

preferences.text_cps = 0

Скорость отображения текста. 0 — бесконечная, в противном случае это количество символов в секунду для отображения. Эквивалент настройки «скорость текста».

preferences.transitions = 2

Определяет, какие переходы должны отображаться. 2 показывает все переходы, 0 не показывает переходов. (1 зарезервировано.) Эквивалент настройки «переходы».

preferences.video_image_fallback = False

Если True, вместо видеоспрайтов отображаются изображения. Если False, видеоспрайты отображаются как обычно. Эквивалент (инвертированный) настройки «видео спрайты».

preferences.voice_sustain = False

Если True, голос продолжает играть до завершения или до тех пор, пока его не заменит другая реплика. Если False, голосовая реплика заканчивается, когда диалоговая строка переходит к следующей. Эквивалент настройки «продолжительность голоса».

preferences.wait_voice = True

Если True, режим автопрокрутки будет ждать завершения голосовых файлов и самоозвучивания перед переходом. Если False, этого не произойдет. Эквивалент настройки «ждать голоса».

preferences.system_cursor = False

Если True, системный курсор будет использоваться принудительно, игнорируя значения config.mouse и config.mouse_displayable. Если False, этого не произойдет. Эквивалент настройки «системный курсор».

preferences.audio_when_minimized = True

Если False, аудиоканалы останавливаются, когда окно сворачивается, и возобновляются, когда окно восстанавливается. Если True, состояние окна не будет влиять на звук. Эквивалент настройки «звук при сворачивании».

preferences.audio_when_unfocused = True

Если False, аудиоканалы останавливаются, когда окно теряет фокус клавиатуры, и возобновляются, когда окно восстанавливает фокус. Если True, фокус клавиатуры не будет влиять на звук. Эквивалент настройки «звук при потере фокуса».

preferences.web_cache_preload = False

Если True, файлы игры будут загружены в кеш веб-браузера, что позволит играть в игру оффлайн. Если False, файлы игры не будут загружены в кеш веб-браузера, и для игры потребуется доступ в Интернет. Эквивалент настройки «предзагрузка веб-кеша».

preferences.voice_after_game_menu = False

Если True, голос будет продолжать воспроизводиться после отображения игрового меню. Если False, голос будет остановлен при отображении игрового меню. Эквивалент настройки «голос после меню».

preferences.restore_window_position = True

Если True, Ren'Py попытается восстановить положение окна при перезапуске игры. Если False, Ren'Py не будет пытаться восстановить положение окна. Эквивалент настройки «восстанавливать положение окна».

preferences.force_mono = False

Если True, Ren'Py смешает стереозвук в моно перед воспроизведением. Если False, стереозвук будет воспроизводиться как обычно.

Настройки микшеров по умолчанию

Эти переменные можно устанавливать только с помощью оператора default.

preferences.volume.main = 1.0

Громкость по умолчанию для главного микшера, которая применяется ко всем каналам в дополнение к индивидуальному микшеру канала. Это должно быть число от 0.0 до 1.0, где 1.0 — полная громкость.

preferences.volume.music = 1.0

Громкость по умолчанию для музыкального микшера, который используется для каналов музыки и видео. Это должно быть число от 0.0 до 1.0, где 1.0 — полная громкость.

preferences.volume.sfx = 1.0

Громкость по умолчанию для микшера sfx, который используется для каналов звуков и аудио. Это должно быть число от 0.0 до 1.0, где 1.0 — полная громкость.

preferences.volume.voice = 1.0

Громкость по умолчанию для голосового микшера, который используется для голосового канала (и, следовательно, для оператора voice, auto-voice и т.д.). Это должно быть число от 0.0 до 1.0, где 1.0 — полная громкость.

Если используются каналы, определённые с помощью renpy.music.register_channel(), громкость по умолчанию этих каналов можно установить с помощью синтаксиса preferences.volume.`mixer`, где mixer — имя микшера.

Например, это установит громкость музыкального микшера на 75%, а микшера sfx — на 50%:

default preferences.volume.music = 0.75
default preferences.volume.sfx = 0.5

Функции микшеров

См. Громкость для получения более подробной информации о микшерах.

preferences.set_mixer(mixer, volume)

Устанавливает mixer на volume.

mixer

Строка, задающая имя микшера. По умолчанию микшеры это "main", "music", "sfx" и "voice" ("main" является специальным микшером).

volume

Число от 0.0 до 1.0, где 0.0 — -40 дБ (по мощности), а 1.0 — 0 дБ (по мощности).

preferences.get_mixer(mixer)

Возвращает громкость для mixer. Если микшер выключен, возвращает 0.0. Возвращает число от 0.0 до 1.0, где 0.0 — -40 дБ (по мощности), а 1.0 — 0 дБ (по мощности).

preferences.set_mute(mixer, mute)

Устанавливает состояние выключения звука для mixer. Если mute равно true, микшер выключается. Если mute равно false, громкость микшера возвращается к значению, которое было до выключения.

preferences.get_mute(mixer)

Возвращает состояние выключения звука для mixer.
