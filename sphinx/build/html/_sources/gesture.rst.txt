.. _gesture:

Жесты
=====

Ren'Py включает в себя распознаватель жестов, который включается при использовании
сенсорного экрана. Это позволяет использовать жесты для функций, которые в противном случае
потребовали бы клавиатуру и мышь.

Распознаватель жестов сначала классифицирует свайпы по восьми направлениям компаса:
"n", "ne", "e", "se", "s", "sw", "w", "nw". Северным считается направление
к верху экрана. Затем он объединяет свайпы в строку, используя "_"
в качестве разделителя. Например, если игрок делает свайп вниз, а затем вправо,
будет создана строка "s_e".

Если :var:`config.dispatch_gesture` имеет значение None, то следующим шагом жест
сопоставляется с событием с помощью :var:`config.gestures`. Если соответствие найдено,
событие ставится в очередь с помощью :func:`renpy.queue_event`. В противном случае
жест игнорируется.

Распознавание жестов включается только тогда, когда "touch" присутствует в
:var:`config.variants`, что обычно и происходит при запуске на устройстве с сенсорным экраном.

.. var:: config.gestures = { "n_s_w_e_w_e" : "progress_screen" }

    Сопоставление жеста с событием, активируемым этим жестом.

.. var:: config.dispatch_gesture : Callable

    Функция, которая используется для обработки жестов. Этой функции
    передаётся необработанная строка жеста. Если она возвращает значение,
    отличное от None, взаимодействие завершается.
