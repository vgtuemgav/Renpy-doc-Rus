Raspberry Pi
============

Ren'Py имеет ограниченную поддержку для запуска на Raspberry Pi. Было
продемонстрировано, что он работает на Raspberry Pi 3B, но может работать и на других моделях.

Важно понимать, что Raspberry Pi — это очень ограниченная система,
даже по сравнению со смартфонами на iOS и Android. В результате не каждая
игра Ren'Py будет хорошо работать — или вообще работать — на Raspberry Pi. Более того,
можно вызвать сбой Ren'Py, позволив игре использовать больше оперативной памяти, чем доступно в системе.

Если Raspberry Pi настроен правильно, Ren'Py должен работать, используя аппаратный OpenGL ES
устройства. Это означает, что он имеет те же ограничения, что и платформы Android и iOS,
в частности, неработающий :propref:`focus_mask`.


Настройка Raspberry Pi
----------------------

Перед использованием Ren'Py следует перенастроить Raspberry Pi с помощью инструмента
raspi-config. Эти настройки находятся в расширенных опциях.

* Memory Split: 256 MB
* Resolution: 1280x720 или меньше
* GL Driver: GL (Fake KMS)

После изменения настроек перезагрузите Raspberry Pi.

Настройка Ren'Py
----------------

Для запуска на Raspberry Pi требуется Arm Linux SDK. Эта версия
Ren'Py включает поддержку всех других платформ и может быть загружена
со страниц загрузок для Ren'Py 7.5 и более поздних версий.

Когда у вас будет ARM-Linux SDK, распакуйте его и перейдите в
созданную директорию.

Запуск игры
-----------

Поскольку Raspberry Pi является платформой с ограниченными ресурсами, мы рекомендуем избегать
использования лаунчера Ren'Py. Вместо этого мы предлагаем использовать renpy.sh напрямую для запуска
проекта, с помощью команды, подобной этой::

    ./renpy.sh /path/to/project

Этот же подход можно использовать для запуска игр Ren'Py, которые напрямую не поддерживают
Raspberry Pi.
