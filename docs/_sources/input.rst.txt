===========
Ввод текста
===========

С некоторыми ограничениями Ren'Py может запрашивать у пользователя ввод небольшого
количества текста. Этот запрос осуществляется функцией :func:`renpy.input`,
которая возвращает введённый текст, позволяя сохранить его в переменной
или обработать иным образом.

В Linux ввод текста ограничен языками, не требующими
поддержки методов ввода (IME). Большинство западных языков должны
работать, а китайский, японский и корейский, скорее всего, нет.

Функция `renpy.input` определяется следующим образом:

.. include:: inc/input

Игры, использующие `renpy.input`, часто требуют дальнейшей обработки
результата с помощью стандартных функций Python для работы со строками. Например,
следующий код запросит у игрока его имя,
удалит начальные и конечные пробелы. Если имя пустое,
оно будет заменено именем по умолчанию. Наконец, оно отображается
пользователю. ::

    define pov = Character("[povname]")

    python:
        povname = renpy.input("What is your name?", length=32)
        povname = povname.strip()

        if not povname:
             povname = "Pat Smith"

    pov "My name is [povname]!"

Здесь длина ввода ограничена 32 символами. Важно
протестировать вашу игру с длинными именами, чтобы убедиться, что эти имена
не нарушают вёрстку текста. В то же время, слишком короткие поля могут
помешать людям ввести предпочитаемое имя.
